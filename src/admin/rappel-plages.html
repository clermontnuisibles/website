📞 Demande de rappel (plages)

Composez un SMS ou message court pour proposer plusieurs créneaux de rappel au client. Le texte se met à jour automatiquement ; enregistrez vos plages par défaut pour les prochaines fois.

Paramètres

Affiché dans le message pour que le client vous rappelle ou réponde par SMS.

Plages horaires

Une ligne par créneau (jour et heures). Ex. : lun. 9h–12h, mer. 14h–17h.

Modèle enregistré.

Message généré
