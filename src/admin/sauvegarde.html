Données stockées localement
Synchronisation Google Drive
Sauvegardez automatiquement sur votre Google Drive et restaurez en un clic. Nécessite un Client ID Google Cloud (configuration unique, gratuite).
⚙️ Configuration requise
  1. Allez sur console.cloud.google.com
  2. Créez un projet → activez l'API Google Drive
  3. Identifiants → Créer → ID client OAuth 2.0 (type : Application Web)
  4. Origines JS autorisées :
  5. Copiez l'ID client (se termine par .apps.googleusercontent.com)
  6. Collez-le dans config.js → champ googleDriveClientId
